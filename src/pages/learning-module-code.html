События в JavaScript

События - это сигналы, которые посылаются при выполнении определенных действий на веб-странице. JavaScript может "слушать" эти события и реагировать на них.

Основные типы событий:

  • click - клик мыши
  • mouseover/mouseout - наведение/уход курсора
  • keydown/keyup - нажатие/отпускание клавиши
  • submit - отправка формы
  • load - загрузка страницы

Добавление обработчиков событий

// Способ 1: addEventListener button.addEventListener('click', function() { console.log('Кнопка нажата!'); }); // Способ 2: onclick свойство button.onclick = function() { console.log('Кнопка нажата!'); }; // Способ 3: HTML атрибут // <button onclick="handleClick()">Нажми меня</button>

Event объект

При срабатывании события, обработчик получает объект события с полезной информацией:

button.addEventListener('click', function(event) { console.log(event.type); // 'click' console.log(event.target); // элемент, на котором произошло событие console.log(event.clientX); // координата X курсора console.log(event.clientY); // координата Y курсора });
